# Iris Classification using K-Nearest Neighbors (KNN)

## Project Overview

This project demonstrates a basic machine learning classification model using the famous Iris dataset. The model is trained using the K-Nearest Neighbors (KNN) algorithm to classify iris flowers into three species based on their measurements.

## Dataset

The project uses the built-in Iris dataset provided by scikit-learn.

- 150 samples
- 3 classes
- 4 features

Features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target Classes:
- Setosa
- Versicolor
- Virginica

## Technologies Used

- Python
- Pandas
- Scikit-learn

## Machine Learning Workflow

1. Load the Iris dataset
2. Prepare the data
3. Split data into training and testing sets
4. Apply StandardScaler
5. Train a KNN classifier
6. Make predictions
7. Evaluate the model using:
   - Accuracy Score
   - Confusion Matrix
   - Classification Report
